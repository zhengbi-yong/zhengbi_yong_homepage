# 欢迎来到 Dioxus 博客

这是一个基于 **Dioxus 0.7.1** 构建的超高性能个人博客。

## 特性

- 🚀 **超高性能**：基于 Rust 和 WebAssembly
- 📝 **Markdown 支持**：完整的 Markdown 渲染
- 🧮 **数学公式**：支持 LaTeX 数学公式
- 📊 **图表支持**：支持 Mermaid 图表
- 🎨 **响应式设计**：适配各种设备
- 🌙 **暗色模式**：支持明暗主题切换

## 技术栈

- **前端框架**：Dioxus 0.7.1
- **语言**：Rust
- **样式**：Tailwind CSS
- **构建工具**：Cargo + Dioxus CLI

## 开始使用

1. 克隆仓库
2. 运行 `dx serve` 启动开发服务器
3. 在 `blogs/` 目录下创建 Markdown 文件
4. 享受高性能博客体验！
